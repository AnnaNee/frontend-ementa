# Placeholder

## O que são
O atributo placeholder não é exclusivo da tag input, mas aqui temos um exemplo dele sendo utilizado com essa tag. Ele irá mostrar um texto, dentro da caixa do `input`, que servirá como referência ao usuário:
`<input type="text" placeholder="Digite seu nome"/>`
