# Nav

## O que é

A tag `<nav></nav>` serve para posicionar elementos de navegação do seu site, isto é, menus.

Exemplo:

```
<header>
  <h1>Título do meu site de presunto</h1>
  <nav>
    <ul>
      <l1>Presunto Sadia</li>
      <li>Presunto Perdião</li>
    </ul>
  </nav>
</header>
```
