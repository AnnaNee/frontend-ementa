# thead

## O que é

A tag `<thead></thead>` serve para representar o cabeçalho da tabela, geralmente composta por células título.

Exemplo:

```
<thead>
  <tr>
      <th>Presunto 1</th>
      <th>Presunto 2</th>
      <th>Presunto 3</th>
  </tr>
</thead>
```
